# Store Assets

Place browser store listing assets in this folder.

## Suggested layout

- `chrome/` - Chrome Web Store screenshots and promo graphics
- `edge/` - Edge Add-ons screenshots and marketing images
- `firefox/` - Firefox Add-ons screenshots and promo graphics
- `shared/` - reusable logos, branding, and source design files

## Capture checklist

- Include wallet create/unlock flow.
- Include send preflight with single/multi note indicators.
- Include transaction lifecycle (pending/retry/confirmed).
- Include mobile sync pairing QR and signed response confirmation.

Do not place secrets, seed phrases, or real wallet identifiers in screenshots.
